# Email Templates

#### Installation
* npm install

* Install MJML extension in visual studio run ext install vscode-mjml.

* For each template make a folder and add index.mjml file

#### Developement

* ctrl + shift + P > mjml:Beautify // to format mjml code

* ctrl + shift + P > mjml:Preview // to preview template

* ctrl + shift + P > mjml:Export Html // to make html file for template

#### MJML Documentation

https://mjml.io/documentation/#mjml-guides
